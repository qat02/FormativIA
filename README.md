# FormativIA Visual

Página web profesional para presentar el GPT **FormativIA Visual**.

## Archivos

- `index.html` — estructura de la página.
- `styles.css` — diseño visual y responsive.
- `script.js` — botones, menú móvil y ventana emergente de ejemplos.
- `assets/logo-formativia.png` — logo del proyecto.

## Cómo usarla

1. Sube todos los archivos a un repositorio de GitHub.
2. En `index.html`, cambia esta línea:

```js
const GPT_LINK = "https://chatgpt.com/g/PEGA-AQUI-TU-LINK-DE-FORMATIVIA";
```

por el enlace real de tu GPT.

3. Activa GitHub Pages:
   - Ve a **Settings**
   - Luego **Pages**
   - En **Branch**, selecciona `main`
   - Carpeta: `/root`
   - Guarda

Tu página quedará publicada con una liga parecida a:

```text
https://tuusuario.github.io/nombre-del-repositorio/
```

## Frase central

Evaluar para mejorar, no para calificar.
